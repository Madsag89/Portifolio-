Flatmate

Mobile Wep App

The concept of flatmate is to match students who rent out their apartment with house seeking students. This is based on their interests and the demands the renter has to their future roomie.

The web app was made using HTML 5, CSS 3, Javascript ES6 and JSON.bin. By using these web technologies, we were able to filter through the diffrent house seekingstudents and only append the ones who matched the renters interests and demands.

Yunik

Website

Yunik is a small jewelry store in the center of Aarhus. The store is very niche and small. Therefore, the store didn’t have the capacity to run a webshop. Also, their existing page was not up to date and was lacking very much regarding best practices and page load time.

Our task was therefore to make a whole new visual identity and make a new and updated website, which also should function as an inspiration universe to convert more people into the physical store.

The website was made using HTML 5, CSS 3, Javascript ES6.

Løgismose

Website

I developed af campagin-site for Løgismose which were called Løgismose Premium. The focus of the page is directed towards truffle-oil.

The main task was to get more attention towards truffle oil and show ways to implement a rather expensive item into the everyday life.

This was done by creating content for the users, show illustrations and all in all make the users more aware of all the benefits of truffle oil.

The website was made using HTML 5, CSS 3, Javascript ES6.
